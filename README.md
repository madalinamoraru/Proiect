# Shadows' Path

## Descriere

Shadow’s Path este un joc de tip platformer cu elemente RPG plasat în Japonia antică, într-o lume în care mitologia și realitatea coexistă. Jucătorul îl controlează pe caracterul principal Shadow, un ninja care pornește într-o călătorie periculoasă. În joc, personajul poate alerga, sări și ataca inamicii care apar pe parcursul nivelurilor. Pe măsură ce avansează, jucătorul întâlnește diferite tipuri de inamici, capcane și obstacole, iar dificultatea crește de la un nivel la altul. De asemenea, pot fi colectate obiecte și power-up-uri care oferă avantaje temporare, cum ar fi viteză mai mare, viață suplimentară sau atacuri mai puternice. (Similar cu Ninja Arashi)
